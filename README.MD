## GCP

### Setup
you will ned to create a `media` folder with this sub foder:

```
/media
+-- audio
|   +-- audio_file.mp3
+-- video
|   +-- /converted
|   +-- /to_convert
+-- transcripts
```

### Run

```
./run.sh gcp
```
